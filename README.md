# MetaShunt Web UI 

Real-time GUI for the MetaShunt V2 power profiler. Features:
- Live **Current vs Time** and **Accumulated Charge vs Time** plots
- **Continuous** or **Burst** measurement modes
- Burst triggers: Immediate, Rising, Falling, Stage, Manual (KEY2)
- **CSV export** (`time [s], current [µA]`)
- **PNG export** of both plots to a chosen folder
